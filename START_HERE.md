## ⚡ Quick Start (5 phút)

### 1. Lấy API Key
Vào: https://console.cloud.google.com/
- Tạo project
- Enable: Maps, Directions, Places APIs
- Tạo API Key

### 2. Setup
```bash
cd frontend
setup-google-maps.bat
# Nhập API key
```

### 3. Run
```bash
npm run build
npm start
```

### 4. Access
http://localhost:3000
- Login: `admin` / `admin`
- Tab: **Monitor**
